TypeScript/JavaScript-Based Telegram Bot
Telegram Advance Order Processing Bot
This bot is highly advanced and intelligent. It receives orders from users, parses them, sends them to the relevant group, and saves them in Google Sheets.

Features:
тЬЕ Webhook setup and checking.
тЬЕ Location detection based on pin code. If the pin code is incorrect, the bot will send the order to the relevant group based on the district name. If there is a spelling mistake in the district name, the bot will automatically correct it and send the order to the group.
тЬЕ Saves orders in Google Sheets.
тЬЕ Sends orders to district-based groups.
тЬЕ Uses Fuzzy Matching to correct input errors.

Functions:

checkWebhook() тЖТ To check the webhook status.
getLocationByPin(pinCode) тЖТ Retrieves location details based on pin code.
fuzzyMatch(str1, str2) тЖТ Checks similarity between two strings.
doPost(e) тЖТ Processes the received message.
saveOrderToSheet(orderDetails) тЖТ Saves the order to Google Sheets.
sendOrderToDistrictGroup(orderDetails, chatId) тЖТ Sends the order to the relevant district group.
sendMessage(chatId, text) тЖТ Sends a message to the user.
